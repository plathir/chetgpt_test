Οδηγίες Χρήσης

  1. Ανοίξτε τη σελίδα στο πρόγραμμα περιήγησής σας.
  2. Πληκτρολογήστε το μήνυμά σας στο πεδίο εισαγωγής.
  3. Πατήστε το κουμπί "Προσθήκη" για να προσθέσετε μια εργασία.
  4. Οι εργασίες εμφανίζονται στη λίστα παρακάτω.
  5. Μπορείτε να ολοκληρώσετε ή να διαγράψετε εργασίες με τα αντίστοιχα κουμπιά.
  6. Πατήστε "Καθαρισμός ολοκληρωμένων" για να διαγράψετε όλες τις ολοκληρωμένες εργασίες.

Για οποιοδήποτε πρόβλημα, επικοινωνήστε με τον διαχειριστή του συστήματος.
