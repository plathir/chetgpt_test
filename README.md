# Todo List (Greek UI)

Ναι — μπορείς να το κάνεις **live test** τοπικά πολύ εύκολα.

## 1) Εκκίνηση τοπικού server
Από τον φάκελο του project:

```bash
python3 -m http.server 4173
```

## 2) Άνοιγμα στον browser
Πήγαινε στο:

- `http://localhost:4173/index.html`

## 3) Τι να δοκιμάσεις
- Προσθήκη νέας εργασίας.
- Ολοκλήρωση/αναίρεση εργασίας.
- Διαγραφή εργασίας.
- Καθαρισμός ολοκληρωμένων.
- Refresh στη σελίδα για να επιβεβαιώσεις ότι οι εργασίες αποθηκεύονται (localStorage).

## 4) Τερματισμός server
Στο terminal πάτα `Ctrl + C`.
