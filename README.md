## Χαρακτηριστικά
- Προσθήκη νέας εργασίας.
- Σήμανση εργασίας ως ολοκληρωμένη / αναίρεση.
- Διαγραφή μεμονωμένης εργασίας.
- Καθαρισμός όλων των ολοκληρωμένων εργασιών.
- Μήνυμα "κενής λίστας" όταν δεν υπάρχουν εργασίες.
- Αποθήκευση εργασιών στο `localStorage` ώστε να μην χάνονται σε refresh.

## Αρχεία έργου
- `index.html`: Δομή UI και styling.
- `app.js`: Λειτουργικότητα Todo List (events, render, localStorage).
- `README.md`: Οδηγίες χρήσης.

## Προαπαιτούμενα
Χρειάζεσαι μόνο ένα από τα παρακάτω:
- **Python 3** (προτεινόμενο), ή
- οποιονδήποτε απλό static server.

Για έλεγχο ότι έχεις Python 3:

```bash
python3 --version
```

## Γρήγορη εκκίνηση (Live Test)

### 1) Πήγαινε στον φάκελο του project

```bash
cd /workspace/chetgpt_test
```

### 2) Ξεκίνα τοπικό server

```bash
python3 -m http.server 4173
```

Θα δεις μήνυμα τύπου:

```text
Serving HTTP on 0.0.0.0 port 4173 ...
```

### 3) Άνοιξε την εφαρμογή στον browser

- `http://localhost:4173/index.html`

### 4) Σταμάτησε τον server

Στο terminal που τρέχει ο server, πάτησε:

```text
Ctrl + C
```

## Αναλυτικός έλεγχος λειτουργιών
Με την εφαρμογή ανοιχτή στον browser, δοκίμασε τα παρακάτω:

1. **Προσθήκη εργασίας**
   - Γράψε κείμενο στο input.
   - Πάτα `Προσθήκη`.
   - Επιβεβαίωσε ότι εμφανίζεται νέο item στη λίστα.

2. **Ολοκλήρωση / Αναίρεση**
   - Πάτα `Ολοκλήρωση` σε ένα item.
   - Επιβεβαίωσε ότι το κείμενο διαγράφεται οπτικά (line-through).
   - Πάτα `Αναίρεση` για επιστροφή στην αρχική κατάσταση.

3. **Διαγραφή item**
   - Πάτα `Διαγραφή` σε μία εργασία.
   - Επιβεβαίωσε ότι αφαιρείται άμεσα από τη λίστα.

4. **Καθαρισμός ολοκληρωμένων**
   - Ολοκλήρωσε 1+ εργασίες.
   - Πάτα `Καθαρισμός ολοκληρωμένων`.
   - Επιβεβαίωσε ότι αφαιρούνται μόνο οι completed εργασίες.

5. **Empty state**
   - Διέγραψε/καθάρισε όλα τα items.
   - Επιβεβαίωσε ότι εμφανίζεται το μήνυμα:
     `Δεν υπάρχουν εργασίες ακόμα.`

6. **Persistence (localStorage)**
   - Πρόσθεσε 1-2 εργασίες.
   - Κάνε refresh (F5 / Cmd+R).
   - Επιβεβαίωσε ότι οι εργασίες παραμένουν.

## Αποθήκευση δεδομένων
Η εφαρμογή αποθηκεύει τις εργασίες στο browser `localStorage` με key:

- `todo-list-items`

Αν θέλεις να κάνεις πλήρες reset δεδομένων:
1. Άνοιξε DevTools (`F12`).
2. Πήγαινε Application/Storage → Local Storage.
3. Διέγραψε το key `todo-list-items`.
4. Κάνε refresh στη σελίδα.

## Συχνά προβλήματα & λύσεις

### Το `http://localhost:4173` δεν ανοίγει
- Έλεγξε ότι ο server τρέχει στο terminal.
- Έλεγξε ότι δεν υπάρχει άλλο process στην ίδια πόρτα.
- Δοκίμασε άλλη πόρτα:

```bash
python3 -m http.server 8080
```

και άνοιξε `http://localhost:8080/index.html`.

### Παίρνω σφάλμα ότι δεν υπάρχει `python3`
- Εγκατάστησε Python 3 ή χρησιμοποίησε άλλο static server.

### Δεν αποθηκεύονται οι εργασίες
- Βεβαιώσου ότι δεν είσαι σε private/incognito mode με μπλοκαρισμένο storage.
- Έλεγξε αν έχεις ενεργοποιημένα extensions που καθαρίζουν αυτόματα localStorage.
